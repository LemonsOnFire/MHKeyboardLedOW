Marvel Heroes LED Lighting Effects

Toggle for Key Color Change by Hero    

Currently selected Hero:
Having this enabled will change the keys used to a color scheme based off of the character you have selected in game.

Toggle for Pulse Keys Red on Hero Death    
Toggle for Flood Keys Red on Hero Death    

When Pulse is enabled all the keys pulse red for 2 seconds then goes back to the previous lighting.
When Flood Keys is enabled the keys will slowly change to red from the top down and then go back to the previous lighting.

Toggle for Flash Keys White on Level Up    

When this is enabled all the keys flash white for a half second when your Hero Levels Up then return to the previous lighting.

Toggle for Flash U key when Ultimate is used    

Flash the U Key when your Ultimate is used.
